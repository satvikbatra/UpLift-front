# Contributing to UpLift Faculty Performance Management System

Thank you for your interest in contributing to UpLift! We welcome contributions from the community.

## Development Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`

## Code Style Guidelines

- Follow the existing TypeScript/React patterns in the codebase
- Use ESLint and Prettier for code formatting
- Write meaningful commit messages following conventional commit style

## Pull Request Process

1. Create a feature branch from `main`
2. Make your changes with clear, focused commits
3. Ensure all tests pass: `npm test`
4. Submit a PR with a clear description of changes

## Issue Reporting

- Use GitHub Issues to report bugs or request features
- Provide clear reproduction steps for bugs
- Include relevant environment details
